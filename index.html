iBook Management System

Overview: The system is a book management platform with two roles: Admin and Customers. Admins can manage book details, set prices, and handle orders, while customers can browse, purchase, and sell books. The platform uses HTML, CSS, JavaScript, Bootstrap for the frontend and Servlets, JSP for the backend, supporting Cash on Delivery (COD) payments.

Key features:

  • Roles: The system has two roles: Admin and Customers.
  • Admin Capabilities: Admin can manage book details, set prices, acknowledge orders, and add or update books.
  • Customer Capabilities: Customers can log in, browse, and purchase books. They can also sell their old books, view their order history.
  • Payment: Currently supports Cash on Delivery (COD) for all orders.
  • Customer Support: Includes functionality to handle user queries efficiently.
  • Technology Stack: Developed using:
    • Frontend: HTML, CSS, JavaScript, Bootstrap
    • Backend: Servlets, JSP
